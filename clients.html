FOR MNOs

One of the greatest challenges facing Mobile Network Operators (MNOs) is how to deliver the best possible mobile experience to subscribers. At any given moment, a user might have access both to cellular networks and Wi-Fi hotspots.

And thus, at any given moment, you must decide which data delivery system will result in the highest subscriber satisfaction.

In the absence of detailed analytics (i.e. mobile devices, buffering speeds, locations), making this decision involves a great deal of guesswork – guesswork that often burdens the network and results in a sub-optimal experience for subscribers.

But what if you could tap into a growing database of actionable analytics built around:

  • User locations
  • User devices
  • Mobile applications
  • Network quality
  • Public Wi-Fi access
  • Speed (through-put)

With these real-time data at your fingertips, you would be able to seamlessly transition subscribers to whatever available networks (Wi-Fi or cellular) deliver the best results at that exact moment in time.

At wefi, we specialize in Intelligent Network Selection, allowing you to deliver optimal results with minimal guesswork. This means fewer costs to you (the MNO), faster load times for end-users, and less congestion across your network.

FOR MSOs
At wefi, we offer a more streamlined approach built around both historic and real-time analytics that include:
  • Public Wi-Fi access
  • Speed (through-put)
  • User locations
  • Mobile devices
  • Network quality

As a MultiScreen Operator (MSO), you want to provide the widest possible coverage using the fewest possible resources. Accomplishing this requires intelligently deploying new Wi-Fi hotspots in high traffic areas – areas where your subscribers congregate in the largest numbers.

But how exactly do you determine where to erect new Wi-Fi hotspots?

Traditional methods involved looking at historic user data and making educated guesses about where to divert limited resources.

With this real-time Intelligent Network Selection, you’re able to deliver a superior user experience for paying subscribers, thus, boosting your retention numbers. Equally important, these real-time data enable you to more effectively deploy limited resources for optimal impact – an approach that reduces operational costs and network loads.

FOR CONTENT PROVIDERS

As a content provider, you build your subscriber base by offering the highest quality, on-demand media possible – regardless of when or where a user logs on.

However, you don’t always have real-time access to user analytics such as:
  • Network loads
  • Mobile devices
  • User locations
  • Wi-Fi availability

Consequently, content delivery requires a great deal of guesswork. And this often results in a poor user experience (for the subscriber) and higher costs (for the content provider):

  • If the bitrate is too low, you make suboptimal use of available bandwidth
  • If the bitrate is too high, users experience unnecessary buffering

And thus, many content providers simultaneously send out multiple versions of the same content. This solution covers all the bases, but it also represents the worst of both worlds. Users receive low quality content AND unnecessary wait-times.



WEFI'S PREDICTIVE ANALYTICS

At wefi, we analyze data across numerous cellular networks and hundreds of millions of Wi-Fi hotspots to provide you with real-time and predictive insights into user experiences.

With these detailed analytics, you know (in advance) the highest possible bitrate that every subscriber in your network can receive – prior to content delivery. And throughout each visitor’s session, our real-time data allow you to adjust the bitrate of that content – even if a subscriber changes locations.

Moreover, we don’t simply analyze user behavior. We also evaluate the quality of all available cellular and Wi-Fi networks. This real-time analysis allows you to adjust the bitrate of your content as subscribers change networks mid-delivery stream.

By leveraging wefi’s predictive analytics and intelligent user offloading, you’re able to offer subscribers the highest acceptable bitrate at the lowest possible cost. The end result is a better user experience for all subscribers, regardless of time, location, or network access.
